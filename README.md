# 🚀 AI-Distiller

Plateforme de création d'assistants IA spécialisés via distillation de modèles.

## ✨ Features

- 🎓 Distillation de modèles puissants (Claude, GPT) vers des modèles locaux
- 🧠 Orchestration avancée (LangGraph, CrewAI, OpenHands)
- 🔍 Capacités étendues (Chain of Thought, Web Search, RAG)
- 🎯 Spécialisation par domaine
- 💰 Optimisation des coûts

## 🚀 Quick Start

```bash
# Installation
pip install ai-distiller

# Assistant interactif
ai-distiller wizard

# Ou configuration manuelle
ai-distiller init --domain support_client
ai-distiller generate --examples 10000
ai-distiller train
ai-distiller serve
```
