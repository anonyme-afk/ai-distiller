# Welcome to AI-Distiller

AI-Distiller is a powerful Python framework that helps you create specialized AI assistants through knowledge distillation from large models (like Claude 3.5 or GPT-4o) to smaller, domain-specific models.

## Key Features

- **Teacher-Student Architecture**: Seamlessly orchestrate powerful LLMs to generate high-quality synthetic training data.
- **Orchestration Integrations**: Native support for LangGraph, CrewAI, and OpenHands.
- **Web & RAG Capabilities**: Endow your distilled agents with web search (Tavily/DuckDuckGo) and document retrieval (LlamaIndex).
- **Interactive UI**: A full Gradio dashboard to monitor fine-tuning and chat with your agents.
