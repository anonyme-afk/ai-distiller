# Contributing to AI-Distiller

First off, thank you for considering contributing to AI-Distiller! 

## Getting Started

1. Fork the repository and clone it locally.
2. Install dependencies: `make install`
3. Set up pre-commit hooks: `pre-commit install`

## Development Workflow

1. Create a branch for your feature or bug fix: `git checkout -b feature/my-feature`
2. Write your code and tests.
3. Run formatting and linting: `make format` and `make lint`
4. Run tests: `make test`
5. Commit your changes and open a Pull Request.

## Pull Request Guidelines

- Ensure all CI tests pass.
- Update documentation if you introduce new features.
- Write a clear and concise PR description.
